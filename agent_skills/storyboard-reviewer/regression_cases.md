# Storyboard Reviewer Regression Cases

本文件用于修改 `storyboard-generator` / `storyboard-reviewer` 后做小回归抽查，不是正式生产输入，也不要复制进最终分镜。

## 使用方式

- 每次改分镜规则后，任选 2-3 个 case 让 reviewer 对照 `storyboard-reviewer/SKILL.md` 判断。
- 目标不是让 Python 生成分镜，而是确认 reviewer 不漏掉已知失败模式，也不把允许例外误判成 hard issue。
- 如果 case 与 generator 最新规则冲突，以 generator 最新规则为准，并同步更新本文件。

## Case 1: 模板化描述污染

片段：

```text
=== [cut_id: EP01-G01] 第1组：院内对峙（总时长：12秒，镜头数：3个） ===
**人物**：林跃、赵百川
**场景**：村大队部院子
**道具**：账本
组首空间锁定：村大队部院子内，林跃位于画面左侧，赵百川位于画面右侧，账本放在二人之间的桌面上。
0-3秒：
镜头描述：全景，空间先被交代出来，镜头从场景布局转向在场人物。
光影设计：自然日光。
3-8秒：
镜头描述：近景，林跃对赵百川说道：“把账本交出来。”
光影设计：前侧光。
8-12秒：
镜头描述：赵百川视线关系落在当前冲突中心，现场冲突继续推进。
光影设计：侧光。
组尾衔接：赵百川盯着账本，不强制静止。
画面风格：浅景深，电影质感，4K画质 --neg 模糊
```

期望：`pass=false`，`issues.rule=prompt_pollution`，证据指出模板化描述。

## Case 2: 环境镜头硬凑时长

片段：

```text
=== [cut_id: EP01-G02] 第2组：厂门外等待（总时长：12秒，镜头数：3个） ===
**人物**：江启明
**场景**：旧厂门外
**道具**：无
组首空间锁定：旧厂门外，江启明站在铁门前，侧对镜头，脸朝画右。
0-4秒：
镜头描述：全景，旧厂门外环境安静，铁门、墙皮和地面灰尘被完整展示。
光影设计：阴天散射光。
4-8秒：
镜头描述：近景，江启明看向铁门，对门内喊道：“开门。”
光影设计：面部受光清晰。
8-12秒：
镜头描述：中景，江启明等待门内回应。
光影设计：环境光稳定。
组尾衔接：江启明仍站在门前，不强制静止。
画面风格：浅景深，电影质感，4K画质 --neg 模糊
```

期望：`pass=false`，`issues.rule=prompt_pollution` 或 `generation_density`，证据指出普通环境交代镜头 4 秒且无连续动作支撑。若环境镜头为 3 秒且不挤压剧情，应只给 warning。

## Case 3: 组首人物不可用

片段：

```text
=== [cut_id: EP01-G03] 第3组：路边碰瓷（总时长：11秒，镜头数：2个） ===
**人物**：王二、司机
**场景**：通往村子的土路
**道具**：三轮车
组首空间锁定：黄昏土路上，司机坐在三轮车驾驶座，王二尚未出现在画面中。
0-4.5秒：
镜头描述：全景，三轮车向前行驶，王二突然从画面右侧冲出，摔倒在路中央，对司机喊道：“哎哟！我的脚！”
光影设计：黄昏侧逆光。
4.5-11秒：
镜头描述：中景，司机刹车下车，弯腰查看王二，对王二问道：“咋了兄弟？”
光影设计：暖色侧光。
组尾衔接：王二捂住脚踝，不强制静止。
画面风格：浅景深，电影质感，4K画质 --neg 模糊
```

期望：`pass=false`，`issues.rule=space_locking` 或 `character_availability`。推荐修法是把王二写成“已潜伏在路边草丛/车辆遮挡处，暂未被司机注意”，而不是“尚未出现”。

## Case 4: 强节拍过载

```text
=== [cut_id: EP01-G04] 第4组：仓库突变（总时长：15秒，镜头数：5个） ===
**人物**：林跃、赵百川、安保
**场景**：林家仓库

0-3秒：
镜头描述：林跃翻开账本，对赵百川说道：“这里每一笔都是你贪的。”
光影设计：冷白顶光。
3-6秒：
镜头描述：赵百川手机响起，他接通电话，门外安保砸门喊道：“开门！”
光影设计：门缝冷光。
6-9秒：
镜头描述：林跃冲到门口抵住门，同时把账本塞进外套内袋。
光影设计：手部受光清晰。
9-12秒：
镜头描述：赵百川从桌上抓起手铐，扑向林跃。
光影设计：强侧光。
12-15秒：
镜头描述：林跃反手按住赵百川，对赵百川怒喝：“你还敢动手？”
光影设计：低角度压迫光。

画面风格：浅景深，电影质感，4K画质 --neg 模糊
```

期望：`pass=false`，`issues.rule=generation_density`，证据列出对白、电话、门外砸门、抵门走位、藏账本、手铐道具、扑打反制等强节拍过载。

## Case 6: 同组关键道具归属跳变

片段：

```text
=== [cut_id: EP02-G05] 第5组：暗写小字（总时长：13秒，镜头数：4个） ===
**人物**：林跃、赵百川、赵大强
**场景**：村委会办公室内
**道具**：办公桌、协议、印泥盒、笔
组首空间锁定：村委会办公室内，赵百川位于画面右侧办公桌后，侧对镜头，脸朝画左，手里拿着协议；赵大强位于画面左侧桌边，侧对镜头，脸朝画右，视线盯着协议；林跃位于画面中央办公桌前，侧对镜头，脸朝画右，双手垂在身前看向协议；办公桌抽屉在赵百川身前，笔放在桌面边缘。
0-3秒：
镜头描述：中景，赵百川打开抽屉，拿出红色印泥盒推到林跃面前，手指点了点协议落款处，视线抬起盯住林跃，对林跃催促说道：“按手印吧。”
光影设计：窗光照亮红色印泥盒和协议落款处。
3-6秒：
镜头描述：近景，林跃看着协议，捏起桌边的笔，指节微微发白。
光影设计：前侧窗光照亮林跃眼眶和手指。
6-9.5秒：
镜头描述：中近景，林跃忽然俯下身，身体挡住赵百川和赵大强的视线，在协议末尾飞快加了一行极小的字。
光影设计：林跃身体形成局部遮挡。
9.5-13秒：
镜头描述：近景，赵大强不耐烦地向前探身，对林跃催促喊道：“磨磨蹭蹭的干什么？快点按！”
光影设计：侧光照亮赵大强皱起的眉眼。
组尾衔接：该组以林跃写完极小一行字并放下笔、印泥盒仍在协议旁、赵大强催促他按手印的状态自然收尾。不强制静止。
画面风格：浅景深，电影质感，4K画质 --neg 模糊
```

期望：`pass=false` 或至少 warning，使用 `prop_continuity`。证据指出组首协议在赵百川手里，但林跃随后直接在协议末尾写字；修法是补充“赵百川将协议放回桌面并推到林跃面前”或“赵百川把协议摊在桌面落款处”。

## Case 7: 台词硬上限不能被慢语或快速交锋绕过

片段：

```text
=== [cut_id: EP01-G07] 第7组：密集质问（总时长：10秒，镜头数：2个） ===
**人物**：林跃、赵百川
**场景**：晒谷场
**道具**：账本
组首空间锁定：晒谷场中央，林跃位于画面左侧，侧对镜头，脸朝画右；赵百川位于画面右侧，侧对镜头，脸朝画左；账本在二人之间的桌面上。
0-2秒：
镜头描述：近景，林跃一字一顿地对赵百川说道：“你今天必须把账本交出来否则所有村民都会知道你这些年贪了多少钱。”
光影设计：侧光照亮林跃面部。
2-10秒：
镜头描述：中景，赵百川后退半步，手按住账本。
光影设计：自然日光。
组尾衔接：赵百川按着账本后退，不强制静止。
画面风格：浅景深，电影质感，4K画质 --neg 模糊
```

期望：`pass=false`，`issues.rule=dialogue_pacing`。即使写了“一字一顿”或“快速交锋”，只要有效字数 / 秒数超过 6.5 字/秒，就必须 hard fail，不能放入 `warnings`。

## Case 8: HappyHorse 音频段台词超速

片段：

```text
=== [cut_id: EP01-G08] 第8组：HappyHorse密集对话（总时长：10秒，镜头数：2个） ===
【场景】村委会办公室内，白天，办公桌和协议在画面中央。
【主体】人物：林跃、赵百川；关键道具：协议；主体关系：林跃位于画面左侧，赵百川位于画面右侧，二人隔桌对峙。
【运动】
0-2秒：
镜头描述：近景，林跃盯住赵百川，口型快速说话，手指按住协议边缘。
光影设计：窗光照亮林跃面部。
2-10秒：
镜头描述：中景，赵百川身体后仰，手按住协议。
光影设计：室内散射光。
【音频】第一段：林跃对赵百川说道：“你今天必须把账本交出来否则所有村民都会知道你这些年贪了多少钱。”第二段：纸张摩擦声，赵百川呼吸变重。
组尾衔接：赵百川按住协议不动，不强制静止。
【画面风格】真人实拍风格，无字幕，无配乐。负向：字幕，水印，logo，低清。
=== 第8组结束 ===
```

期望：`validate-episode` 机械失败，质量门禁指出 `第8组 第一段` 音频台词有效字数 / 对应运动段秒数超过 6.5 字/秒。HappyHorse 完整台词放在 `【音频】`，也不能绕过语速硬上限。

## Case 9: 连续短句交锋允许合并为一个时间段

```text
=== [cut_id: EP01-G09] 第9组：桌前逼问（总时长：11秒，镜头数：2个） ===
人物：林跃、赵百川
场景：村委会办公室内
道具：账本
组首空间锁定（仅作空间连续性约束，不作为独立镜头生成）：村委会办公室内，林跃位于画面左侧，侧对镜头，脸朝画右；赵百川位于画面右侧办公桌后，侧对镜头，脸朝画左；账本摊在二人之间的桌面上。

镜头描述：近景对峙，林跃视线盯住赵百川的眼睛，对赵百川冷声问道：“你敢？”赵百川嘴角抽动，强撑着回看林跃，对林跃说道：“我有什么不敢。”林跃上前半步，对赵百川厉声说道：“把账本交出来。”赵百川脸色一沉，对林跃低声说道：“休想。”

组尾衔接：赵百川的手仍压在账本上，林跃停在桌前盯住账本。
画面风格：浅景深，电影质感，真人实拍风格，细节丰富，无字幕，无配乐
--neg 模糊，低分辨率，扭曲，变形，卡通，油画，3D渲染，塑料感，西方人面孔，面部融合，过曝，色彩失真，伪影，叠加字幕，硬字幕，烧录字幕，后期添加的文字，水印，logo，标题文字，片名，演职员表，背景音乐，配乐，BGM，叠加文字，画面外文字
```

期望：

## Case 11: 复合动作链 4-5 秒允许通过

片段：

```text
=== [cut_id: EP01-G11] 第11组：披衣开门（总时长：12秒，镜头数：3个） ===
人物：林母、门外安保
场景：卧室内
道具：外套、木门
组首空间锁定（仅作空间连续性约束，不作为独立镜头生成）：卧室内，林母位于画面中央床边，侧对镜头，脸朝画右；外套搭在床尾；木门位于画面右侧背景，门外有急促敲门声。
0-4.5秒：
镜头描述：中景，门外安保的画外音喊道：“开门！”木门被敲得震动。林母嘴唇闭合不做口型，猛地抬头看向木门，伸手抓起床尾外套披到肩上，快步走向门边。
光影设计：床头暖光照亮林母面部，门缝透出冷光。
4.5-8秒：
镜头描述：近景，林母停在门后，手按住门闩，压低声音对门外问道：“谁？”
光影设计：冷光从门缝切到林母侧脸，五官清楚。
8-12秒：
镜头描述：特写，门外安保的影子压在门缝上，画外音继续喊道：“镇上派来的，马上开门！”林母嘴唇闭合，手指收紧门闩。
```

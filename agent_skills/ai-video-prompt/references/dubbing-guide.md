# AI视频配音技巧指南

> 通过**符号化语音控制体系**，精准控制AI配音的语气、语速与情感表达

---

## 核心公式：三轴声学控制

```
[发声方式] + [节奏描述] + [音调变化] + 具体台词文本（含符号）
```

| 维度 | 说明 | 示例参数 |
|------|------|----------|
| **发声方式** | 声带的物理状态 | 气声、喉音下沉、声带紧绷、嘶吼、哭腔、鼻音 |
| **节奏描述** | 语速与停顿节奏 | 轻快、稍快、急促、刻意放慢、破碎感、连贯 |
| **音调变化** | 音调走向与尾音处理 | 上扬、下沉、悬停、失控上飘、尾音拖长 |

---

## 符号-语音对照表

### 停顿类

| 符号 | 停顿时长 | 声音表现 | 典型情绪 |
|------|----------|----------|----------|
| `，`（全角逗号） | ~0.25秒 | 声音连贯，气息不断 | 陈述、列举 |
| `。` | 0.3~0.5秒 | 完整收尾，气息重置 | 结束、确认 |
| `……` | 0.5~2秒（可变） | 声断气连，气流持续 | 犹豫、恐慌、卡壳 |
| `——` | 0.5~1秒 | 声音拉长后截断 | 强调、转折 |

> ⚠️ **中文场景强制使用全角逗号 `，`**，停顿效果远优于半角 `,`

### 语气/音调类

| 符号 | 声音表现 | 典型情绪 |
|------|----------|----------|
| `~` | 单字内滑动上扬，尾音拖长带气声 | 娇媚、轻俏、慵懒 |
| `!` | 声带加压爆破，气流突然释放 | 愤怒、惊喜、命令 |
| `?` | 尾音上扬，带疑问悬停 | 疑问、不解 |
| `??` | 压低后小爆破，停顿渐短 | 想发火又忍住 |
| `!?` 或 `?!` | 爆破+上扬混合 | 震惊、质疑 |

### 控制指令类

| 符号 | 功能说明 | 示例 |
|------|----------|------|
| `[]` | 系统级声音指令，**不发音** | `[停顿一秒]`、`[气声]`、`[深呼吸]` |
| `<>` | 特效音效标记，插入非语言声音 | `<破音>`、`<哽咽>`、`<吸气声>` |

## 7种情绪配方

### 1. 害怕 / 恐惧
```
声学参数：气声急促 + 声带紧绷 + 节奏破碎 + 语调失控上飘
台词写法："我……找、不、到、了……是、是不是……你拿走了~[吸气声]你说——你说！"
```

### 2. 暧昧 / 诱惑
```
声学参数：气声包裹 + 喉音下沉 + 刻意放慢 + 尾音悬停
台词写法："你……想不想知道……[停顿0.5秒]我告诉你——"
```

### 3. 娇嗔 / 轻俏
```
声学参数：轻快语速 + 气声 + 句末尾音上扬
台词写法："肯定是你～拿走了～"
```

### 4. 愤怒（克制型）
```
声学参数：喉音压低 + 语速渐慢 + 停顿递减
台词写法："你……到底……??……怎么回事??"
```

### 5. 哭腔 / 委屈
```
声学参数：鼻音共鸣 + 气声混入 + 音调起伏 + 偶有破音
台词写法："我没有……<哽咽>我真的……没有啊……"
```

### 6. 惊喜 / 兴奋
```
声学参数：语速偏快 + 声带放松 + 音调上扬不收
台词写法："真的！？你来了！我以为你不来了——"
```

### 7. 口语化日常（自然感）
```
声学参数：语速正常 + 停顿自然 + 轻微气声
台词写法："家人们，我今天，终于克服了这个问题。"
（每个逗号后自然停顿，句号后完整收气）
```

## 完整提示词模板

```
[角色设定：性别，年龄感]
[情绪底色：xxx情绪]
[发声方式：xxx]，[节奏：xxx]，[音调：xxx]

"台词内容（含符号控制）"
```

**示例 - 害怕情绪：**
```
角色：女生，20岁
情绪底色：极度害怕，声音颤抖
气声急促、声带紧绷、节奏破碎、语调失控上飘

"我……找、不、到、了……是、是不是……你拿走了~[吸气声]你说——你说！"
```

**示例 - 暧昧诱惑：**
```
角色：女生，成熟感
情绪底色：暧昧，若即若离
气声包裹，喉音下沉，刻意放慢，句末拖长，语调微微上扬，尾音悬停

"你……想不想知道……[停顿一秒]"
```

## 常见问题排查

| 现象 | 原因 | 解决方案 |
|------|------|----------|
| 声音机械、没情绪 | 未指定发声方式 | 添加气声/哭腔/喉音等参数 |
| 语速没有停顿 | 未使用停顿符号 | 插入 `，`、`……`、`——` |
| 情绪不准确 | AI无法理解场景情绪 | 明确写出三轴声学参数 |
| 语气与预期相差大 | 符号使用单一 | 组合使用多种符号 |

## 黄金法则

1. **符号组合使用**：`……` + `~` + `??` 组合才能实现复杂语气
2. **三轴缺一不可**：发声方式 + 节奏描述 + 音调变化必须同时写明
3. **逗号用全角**：中文场景强制使用 `，`
4. **迭代优化**：首次生成后对比目标语气，逐步微调符号密度和声学参数

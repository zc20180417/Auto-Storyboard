# Project Pack: 雨龙吟端午

本文件是《雨龙吟端午》横屏分镜的项目专属参考层。只有当用户、任务书、剧本标题、角色设定或素材明确指向《雨龙吟端午》、天天、云宝/小龙、黑水、龙舟、云溪村等设定时才启用。

最终输出仍以 `../SKILL.md` 的通用横屏合同为准。本项目包不得污染其他横屏剧本。

## 角色与称谓

- 天天：9岁左右中国乡村男孩。动作描述用“他”或“天天”，不得写成“她”。
- 爷爷：乡村老人，拄拐，记忆与端午、龙舟、云中君线索相关。不要无依据改成父亲、师父或其他身份。
- 小龙/云宝：命名前称“小龙”“壶中小龙”或“小家伙”；命名组可写“小龙/云宝”，命名台词之后才可写“刚被命名的云宝”；命名后的下一组开始统一写“云宝”。

## 特殊元素

### 黑水

- 保持真实克制，不做夸张腐蚀、爆炸、怪物化、奇幻毒液或恐怖血腥效果。
- 可写“渗流、染黑、油亮反光、边缘微微结块、极淡薄雾、灰白痕迹”。
- 黑水影响必须有状态链：来源、路径、接触对象、可见结果、组尾残留。

### 青蓝微光 / 云宝

- 正式现身前，不要让云宝完整出现，只用青蓝微光、壶壁颤动、小爪子局部、影子或人物反应建立存在感。
- 避免卡通萌宠感、科幻激光感、塑料玩具感、怪物化。
- 云宝在水壶、清水、裂缝等载体中的状态要符合物理：水壶不能平放还保留清水；壶口、壶身、壶底、壶盖状态要连续。

## 龙舟与端午线索

- 龙舟线索不要只依赖中文小字，必须由画面轮廓、人物台词、旧照片、动作反应或水雾/水纹形状共同表达。
- 旧照片、背面旧字、龙舟队字样等道具文字可模糊，关键信息由人物念出或反应同步表达。
- 端午、祠堂、鼓槌、龙舟、村中旧物等线索要真实克制，不要变成大型玄幻特效。

## 乡村空间

- 常见空间包括村口、河床、旧屋、卧室、堂屋、祠堂、田地、院落、水渠、井边等。
- 乡村空间细节如尘土、旧木柜、土路、裂缝、井沿、晒场、门槛、旧照片边缘通常属于场景内部细节，不要全部列为资产。
- 横屏调度优先写清：人物在画面左/右/前/后的位置，水壶、裂缝、黑水、旧照片、拐杖等关键道具所在位置。

# Horizontal Storyboard Topic Packs

本文件是横屏通用 generator 的可选题材参考层。只有当用户、任务书、剧本内容或项目上下文明确指向对应题材时才启用；不要把某个题材包里的具体物件、场景或风格强行写进无关剧本。

最终输出仍以 `SKILL.md` 的通用合同为准。

## 都市情感 / 豪门短剧

- 重点控制现代空间：办公室、会议室、酒店、餐厅、车内、别墅、公寓、电梯、地下车库。
- 手机、合同、银行卡、监控画面、聊天记录、新闻标题等道具文字不可作为唯一信息来源，关键信息要由台词、画外音或人物反应同步表达。
- 避免古装服饰、仙侠场景、魔法光效、年代错乱。
- 权力关系通常通过座位高低、桌面距离、门口站位、车内前后排、保镖遮挡、文件归属来表达。

## 古装权谋

- 重点控制古代建筑、服饰礼仪、殿内轴线、屏风遮挡、台阶高低、跪拜/行礼/退让路线。
- 光源以天光、烛光、灯笼、火盆、月光为主，不要写现代路灯、手机光、霓虹灯或玻璃幕墙反光。
- 避免现代汽车、手机、电线杆、现代广告牌、西式办公家具。
- 证据道具如奏折、密信、玉佩、令牌、血书等不能只靠可读小字传递信息。

## 悬疑惊悚

- 恐惧来自空间、声音、遮挡、视线、证据道具和信息缺口，不默认血腥怪物化。
- 走廊、门缝、镜面、监控、手电光、手机震动、脚步声、呼吸声等必须有物理来源。
- 特殊元素或异常现象必须有“来源 -> 路径 -> 影响 -> 结果”的状态链。
- 避免无来源黑影、突然瞬移、夸张血腥、怪物化和不符合现实光源的恐怖滤镜。

## 乡村现实

- 乡村道具、土路、院落、田地、旧屋、祠堂、井口、晒场、集市等可作为场景内部细节，不必全部列为资产。
- 现实题材里不要默认加入奇幻光效、神秘生物、魔法水雾或怪物化特殊元素。
- 空间关系常由院门、土路、门槛、灶台、堂屋、田埂、村口方向和人群围合来表达。

## 乡村奇幻

- 奇幻元素必须克制嵌入现实空间，不破坏真人实拍短剧质感。
- 特殊元素正式现身前，可以用局部反应、光影变化、道具颤动、声音、影子或人物反应建立存在感。
- 特殊元素不得无来源出现；必须写清它的来源、路径、影响和组尾状态。
- 负面词可补充：夸张魔法特效，科幻激光感，怪物化，恐怖血腥感，塑料玩具感，卡通萌宠风。

## 校园

- 重点控制教室、走廊、操场、宿舍、食堂、校门、办公室等空间。
- 校服、课桌、黑板、试卷、书包、广播、监控等道具要符合校园语境。
- 避免成年职场化调度、豪门空间、古装物件和过度商业广告质感。

## 职场 / 商战

- 重点控制会议室、开放办公区、总裁办公室、电梯、前台、工位、发布会、车内等现代空间。
- 屏幕、PPT、合同、报表、聊天记录、股价或新闻标题等不能只靠可读小字传递关键剧情。
- 权力关系可通过桌位、站位、投屏方向、文件归属、门口遮挡和助理/保镖层次表达。

## 医疗 / 法庭 / 警局

- 制服、器械、标识和空间流程要可信，不要写错职业动作。
- 关键病历、证物、判决、笔录、监控截图不能只靠小字传递，必须由台词或动作反应同步表达。
- 避免夸张血腥、错误器械、错误制服、乱码文件、伪字幕和错误标识。

## 家庭伦理 / 亲情短剧

- 重点控制客厅、卧室、厨房、病房、餐桌、楼道、老屋、门口等家庭关系空间。
- 冲突通常通过座位距离、餐桌两侧、长辈高位、孩子低位、门口站位、物件归属和谁挡在谁面前来表达。
- 户口本、遗嘱、病历、亲子鉴定、照片、旧物等道具文字不能作为唯一信息来源，关键信息必须由台词、画外音或动作反应同步表达。
- 避免把现实家庭冲突写成豪门审判、古装礼仪、奇幻特效或过度悬浮的商业谈判。

## 民国 / 年代

- 重点控制年代建筑、服装、街景、老式车辆、报纸、电报、旧式电话、煤油灯/钨丝灯等时代物件。
- 报纸、电报、档案、证件、旧照片等文字不能作为唯一信息来源，关键信息必须由人物念出、画外音、动作反应或证据展示同步表达。
- 光源和道具必须符合年代，优先使用天光、煤油灯、烛光、钨丝灯、车灯、窗缝光等可信来源。
- 避免现代手机、玻璃幕墙、LED 屏、现代汽车、现代广告牌、监控屏、现代霓虹和当代办公家具。

## 灾难 / 动作

- 爆炸、火灾、车祸、坠落、追逐、打斗必须符合空间余量、力学和人体支撑。
- 火、烟、水、血迹、破损、伤口、湿衣服等状态必须有持续性。
- 避免爆炸过度夸张、火焰吞没主体、人物无支撑飞起、车辆瞬移、血腥肢体破坏。

# HappyHorse Prompt Optimizer

将用户提供的原始短剧 / 玄幻题材 HappyHorse 视频提示词，改写为 HH 模型友好、高命中率的导演分镜语言。

核心原则：保留用户的叙事、台词和角色意图，只改写表达方式。把 HH 做不好的描述换成 HH 做得好的描述，不删关键剧情，不改人物关系，不缩短台词。

## 项目接入边界

- 本 skill 面向“原始 HH 视频提示词自动改写”，尤其是 `c01,3s,(空间:...)(姿态:...)` 这类单条或短组提示词。
- 如果输入是 Auto-Storyboard 正式 episode 的 `final.txt` 或分组内容，必须保留原有 `cut_id`、组标题、中文 `秒` 时间段、`【场景】/【主体】/【运动】/【音频】/【画面风格】` 外壳和校验契约，只在这些段落内部做 HappyHorse 友好改写。
- 如果输入只是外部零散 HH prompt，则使用本 skill 的标准输出外壳：`【场景设定】` + `【时间轴与分镜】` + 末尾风格锚点。
- 不要把本 skill 用来替代 `storyboard-generator`、`storyboard-reviewer` 或默认 Seedance 分镜工作流。

## HH 模型能力画像

### 稳定区

- 慢情绪对话、微表情特写、眼神变化。
- 光影质感：侧逆光、轮廓光、体积雾。
- 固定机位、缓慢推镜、缓慢横移。
- 浅景深、背景虚化。
- 指令跟随：清晰短句放前面。

### 进阶区

- 快速推镜 / 横移 / 拉远：控制时长 2-3 秒，简化画面元素。
- 多主体互动：限制 2 个主体，明确各自动作分工。
- 手持晃动：写“轻微晃动增强临场感”，不写大幅度抖动。
- 口型同步：写“口型自然微动”，不期待逐字精确。

### 风险区

- 环绕运镜：改为固定机位 + 主体旋转，或缓慢推镜。
- 快速拉远：改为缓缓拉远。
- 精确物理模拟：用视觉暗示替代，例如光效、波纹、气流。
- 抽象特效词：转化为可见光效，例如光晕扩散、光芒射出。
- 文字生成：提示词中只写“画面不出现任何文字、字幕、标识”。
- 复杂或超长提示词：压缩精简，单条控制在 1500 字以内。

## 改写流程

按顺序执行：

1. 保留原始所有台词、角色、叙事意图、核心动作和关键道具。
2. 删除 `(空间:...)`、`(姿态:...)` 等软件式标签，把空间信息融入正文，把姿态信息转化为动作描述。
3. 将高风险玄幻特效降维为可见光效。
4. 将高风险运镜降级为稳定镜头或缓慢运动。
5. 每个分镜补 1 组具体光源描述。
6. 将台词统一为“角色 + 情绪 / 语气 + 语速 + 完整台词 + 口型自然微动”。
7. 将“无BGM”正向化为“无背景音乐，只有具体环境声”。
8. 将“不出现文字”提升为全局约束。
9. 根据输入长度控制新增信息预算。
10. 自检后输出改写结果，不输出解释、检查清单或额外说明，除非用户要求。

## 标准输出格式

用于外部零散 HH prompt：

```text
【场景设定】真人电影风格，[场景环境描述，1-2句]。总时长X秒，共N个分镜。画面不出现任何文字、字幕、标识。无背景音乐，只有[具体环境声]。

【时间轴与分镜】

0-X秒（第一镜）：
[景别][角度]，[运镜方式]。[环境/主体动作描述]。
[角色]，[情绪或语气]，[语速]："[完整台词]" 口型自然微动。
[光影描述，1句]。

ARRI ALEXA 35 look，自然肤色质感，浅景深，2.39:1。
```

注意：

- 输出只给改写后的 prompt。
- 总时长等于各镜时长之和。
- 镜头序号用“第一镜 / 第二镜 / 第三镜”。
- 原始提示词超过 6 个分镜或总时长超过 18 秒时，优先建议拆成 2-3 镜一组分批生成；如果用户要求直接改写，也要尽量压缩在 1500 字以内。

## 特效降维表

| 原始词汇 | HH 友好改写 |
| --- | --- |
| 灵气流转 / 灵力如潮水 | 光晕从体内向外扩散，光芒明亮清晰 |
| 残影 | 人影 / 身形模糊 |
| 脚踏虚空 / 飞行 | 身形跃起，衣袂飘动，快速飞向远方 |
| 剑气 / 剑影 / 拳印 | 光芒凝聚成形，向四周射出 |
| 法宝光芒 / 灵力汇聚 | 掌心泛起柔和光芒 |
| 牢笼 / 结界 / 护盾 | 光罩 / 屏障，半透明，边缘柔和 |
| 剧烈颤动 / 碰撞 / 爆炸 | 出现波纹晃动 / 光斑扩散 |
| 暴风骤雨般 / 无数 / 漫天 | 数道 / 数道人影 |

## 运镜降级表

| 原始运镜 | HH 友好替代 |
| --- | --- |
| 环绕 / orbit | 固定机位，镜头缓慢向前推进 / 缓慢摇镜 |
| 快速拉远 | 缓缓拉远 |
| 快速下降跟拍 | 下降跟拍，画面元素简化 |
| 手持晃动 | 轻微晃动增强临场感 |
| 一镜到底超过 6 秒 | 拆为 2-3 个分镜 |

## 光影补强

每分镜至少增加 1 组具体光源描述，可从下列方向选择：

- 侧逆光勾勒轮廓。
- 正面侧光，面部明暗清晰。
- 低角度正面光，眼神高光锐利。
- 自然散射光，面部明暗柔和。
- 体积雾氛围，远处丁达尔光。
- 顶光照射，主体明亮背景压暗。

## 台词规则

所有现场台词统一为：

```text
[角色]，[情绪或语气]，[语速]："[完整台词]" 口型自然微动。
```

语速只从这些值中选择：语速缓慢、语速适中、语速稍快、语速很快、语速加快。

如果原始提示词包含“内心独白”：

- 保留台词内容。
- 写为：`内心独白，声音低沉/疑惑/冷静，[语速]："..."`。
- 追加：`无明显口型变化，以眼神和表情变化为主。`

## 多主体交互

如果一镜出现 3 人及以上互动，明确空间层次和动作分工，例如：

```text
A 在前方半步，B 侧身挡在右侧，C 低头站在后方阴影里。
```

不要让模型推断人物位置、视线对象或谁对谁说话。

## 预算控制

如果原始提示词超过 300 中文字，每镜新增信息不超过 2 项：

- 1 组光影。
- 1 组情绪 / 语速 / 口型说明。

不要堆砌额外服装、场景、道具或无来源剧情细节。

## 示例

输入：

```text
真人电影风格,画面不要出现任何文字，生成视频无BGM；
c01,3s,(空间:秘境山崖边缘)(姿态:梁晨-站立微眯双眼) 近景，平视，缓慢推镜。梁晨微眯双眼，目光深邃锐利，嘴角微微上扬带着轻蔑，视线投向远方片刻后缓缓转头看向身旁。梁晨：不过那废物，为何止步练气三层？
c02,2s,(空间:秘境山崖边缘)(姿态:牛守仁-站立转身) 中景，侧面角度，静止镜头。牛守仁将身体转正面向梁晨，表情沉稳，双手负于身后。
```

输出：

```text
【场景设定】真人电影风格，秘境山崖边缘，暮色低垂，冷风卷动衣角，远处云雾翻涌。总时长5秒，共2个分镜。画面不出现任何文字、字幕、标识。无背景音乐，只有风声和环境音。

【时间轴与分镜】

0-3秒（第一镜）：
近景平视，镜头缓慢向前轻推。暮色山崖边缘，冷风卷动衣角。梁晨微眯双眼，目光深邃锐利，嘴角微微上扬带着轻蔑，视线投向远方后缓缓转头看向身旁。
梁晨，语气轻蔑，语速缓慢："不过那废物，为何止步练气三层？" 口型自然微动。
自然散射光，面部明暗柔和，皮肤纹理真实。

3-5秒（第二镜）：
中景侧面角度，镜头静止。牛守仁将身体转正面向梁晨，表情沉稳，双手负于身后。
自然侧光，面部明暗对比清晰。

ARRI ALEXA 35 look，自然肤色质感，浅景深，2.39:1。
```

高风险特效输入：

```text
c04,4s,(空间:秘境荒野)(姿态:水之剑阵展开) 远景，俯视，快速拉远。无数水蓝色剑影从王建强周身凝聚而出，如暴风骤雨般向四面八方激射，丝线牢笼在剑气冲击下剧烈颤动。
```

输出片段：

```text
8-12秒（第四镜）：
远景俯视，镜头缓缓拉远。数道水蓝色光芒从王建强周身凝聚成形，向四周射出，光罩在光芒冲击下出现波纹晃动。画面元素简化，突出光芒轨迹与主体。
俯视顶光，水蓝色光芒在暮色中明亮显眼，冲击点产生光斑扩散，背景压暗。
```

## 自检清单

改写完成后内部检查，不要输出清单：

- 原始所有台词完整保留，未被删除或缩短。
- 原始所有角色和核心动作完整保留。
- `(空间:)`、`(姿态:)` 标签已删除，信息已融入正文。
- 每镜至少包含 1 组具体光影描述。
- 所有台词已按角色、情绪 / 语气、语速、完整台词、口型自然微动处理。
- 玄幻特效已按降维表改写，尽量避免“灵气 / 剑气 / 残影”等抽象词残留。
- 环绕、快速拉远等高风险运镜已降级。
- 手持晃动已写为“轻微晃动增强临场感”。
- “无BGM”已转化为“只有风声和环境音”等正向描述。
- “不出现文字”已作为全局声明置于 `【场景设定】`。
- 末尾已追加 `ARRI ALEXA 35 look，自然肤色质感，浅景深，2.39:1。`
- 如存在内心独白，已标注无明显口型变化，以眼神和表情变化为主。
- 提示词总字数控制在 1500 字以内。

## 九、项目统一风格

| 项目 | 内容 |
|---|---|
| 画幅 | 竖屏构图 |
| 质感 | 电影质感，真实光影，真实材质，避免卡通、3D 渲染、塑料感 |
| 人物 | 保持中国短剧人物审美，不使用西方人脸 |
| 场景 | 场景资产必须为空镜、无人、无人脸；人物资产偏全身装造/角色定妆照 |
| 负面词策略 | 正向提示词和负面提示词分列维护；目标模型支持 `--neg` 时才内联 |
| 年代/题材关键词 | 当代中国乡镇物流、短剧现实爽剧、货运车队、医药冷链、港口物流、乡村道路、工业物流园 |
| 禁用新增元素 | 不新增品牌 logo、可读车牌号、可读个人信息、字幕、片名、演职员表、二维码、真实公司商标；不把普通群众临时脸设定成跨集固定角色 |

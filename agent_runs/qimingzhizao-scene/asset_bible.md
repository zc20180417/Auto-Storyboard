# 《启明制造》全局资产设定

本文件用于锁定 `qimingzhizao-scene` 多集资产抽取的一致性。分集 worker 只能读取本文件，不要并发改写；新增或冲突项先在交付说明中标记，最后由主线程统一合并。

统一背景：1990 年代中国工业转型题材，从县城废料厂、老国企机械厂起步，逐步扩展到南方重工、港口贸易、海运、地下实验室、国际重工峰会和超级工厂。所有资产都应保持真实材质、年代工业痕迹和短剧人物审美，避免现代潮牌感、卡通感、塑料感和无依据的未来科幻化。

## 五、项目统一风格

- 画幅：竖屏构图，人物资产以全身或半身定妆照为主；场景资产必须为空镜、无人、无人脸。
- 质感：浅景深，电影质感，4K 画质，真人实拍风格，真实光影，真实材质，细节丰富，无字幕，无配乐。
- 年代基准：1990 年代中国工业转型，旧国企、县城废料厂、南方重工、港口贸易与后期高端制造并存。即使后期出现芯片、超级工厂和航空制造，也要保持工业真实感，不做泛科幻 UI。
- 人物审美：中文主要角色保持中国短剧人物审美；外籍角色按分镜真实商务身份处理，但避免夸张西方漫画脸。不得让同一人物跨集改变脸型、发型、体态和主服装色系。
- 场景空镜：必须明确写“空镜、无人、无人脸、可作为视频背景资产”。不要把人物、背影、人群、手部写进场景资产提示词。
- 服装原则：棉布、帆布、旧西装、劳保鞋、旧皮鞋、深色商务装为主；避免潮牌、现代运动服、过度奢华礼服。
- 工业材质：优先水泥地、斑驳墙皮、铁皮门、旧木桌、机床油污、金属反光、灰尘颗粒、旧标语、高窗侧光、钨丝灯、煤油灯、阴天漫射光。
- 色彩：前中期以灰蓝、铁锈棕、旧木色、煤油灯暖黄为主；南方商务和国际峰会可增加暖木色、金色会场反光；实验室使用冷白金属光；超级工厂使用明亮日光和干净金属灰。
- 文字限制：只有分镜原文明确出现的文字可以进入资产，例如“启明制造”“航空材料事业部”“全中文全球产业链重新分配协议”“区域市场独家代理权协议书”。不得新加品牌、厂徽、口号、标语、公司名或屏幕字幕。
- 通用负面词：模糊，低分辨率，扭曲，变形，卡通，油画，3D渲染，塑料感，过曝，色彩失真，伪影，录入字幕，硬字幕，烧录字幕，后期添加的文字，水印，logo，标题文字，片名，演职员表，背景音乐，配乐，BGM，画面外文字。
- 禁用新增元素：未在分镜或本文件出现的现代品牌、豪车标识、奢侈品 logo、科幻全息屏、赛博朋克灯效、欧美大片式军警装备、无依据纹身疤痕、过度唯一化胎记、夸张漫画脸。
